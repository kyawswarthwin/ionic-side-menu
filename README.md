# Ionic 4 Side Menu

![Screenshot](https://raw.githubusercontent.com/kyawswarthwin/ionic-side-menu/master/docs/img01.png)

## Installation

```sh
npm i ionic-side-menu
```

## Example

```sh
git clone https://github.com/kyawswarthwin/ionic-side-menu.git
cd ionic-side-menu
npm install
ionic serve
```
